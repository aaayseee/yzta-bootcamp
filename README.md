## 4. Sprint Review (Sprint İncelemesi)
* **Tamamlananlar:** Veri setinin entegrasyonu, GitHub repo ve Scrum tahtasının kurulumu, EDA (Keşifsel Veri Analizi) aşaması ve tahmin modelinin (Random Forest) eğitilmesi.
* **Mevcut Durum:** Sprint 1 için planlanan tüm teknik ve idari hedeflere beklenenden daha kısa sürede ulaşılmıştır. Geliştirilen modelin test verisinde %97.4 gibi yüksek bir başarı göstermesi, projenin temelinin oldukça sağlam atıldığını kanıtlamıştır.

## 5. Sprint Retrospective (Sprint Değerlendirmesi)
* **Ne İyi Gitti:** Projenin en başından itibaren disiplinli ve planlı bir şekilde ilerlenmesi, zaman yönetimini kusursuz hale getirdi. Ayrıca Colab ile GitHub entegrasyonu pürüzsüz çalıştı.
* **Neler Zorladı / Geliştirilebilir:** İlk aşamada GitHub reposunu ayağa kaldırırken (initialize) ufak bir yetkilendirme adımı zaman aldı ancak hızlıca aşıldı.
* **Sonraki Sprint İçin Aksiyon Planı:** Modelin arka planı (backend) sağlam bir şekilde oturduğuna göre; bir sonraki aşamada kullanıcıların bu sonuçları görebileceği arayüz (UI/UX) tasarımlarına ve frontend geliştirmelerine ağırlık verilerek proje gerçek bir ürüne dönüştürülecektir.
